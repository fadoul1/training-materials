# GitHub Copilot in Practice

**Configure, Automate, and Extend Your AI Assistant**

Bienvenue dans cette formation de 6 jours pour maitriser GitHub Copilot au-dela des simples suggestions. Vous apprendrez a configurer, automatiser et etendre Copilot pour en faire un assistant de developpement precis, adapte a vos projets professionnels.

## Programme

| Jour | Module | Duree |
|------|--------|-------|
| 1 | [Comprendre Copilot et pourquoi la configuration compte](/modules/module-01) | 1h 00m |
| 2 | [Custom Instructions : regles always-on et file-based](/modules/module-02) | 1h 00m |
| 3 | [Prompt Files : commandes slash reutilisables](/modules/module-03) | 1h 30m |
| 4 | [Agent Skills : automatisation multi-etapes avec scripts](/modules/module-04) | 1h 30m |
| 5 | [Custom Agents : personas specialisees avec scope controle](/modules/module-05) | 1h 00m |
| 6 | [Workflow End-to-End et Projet Final](/modules/module-06) | 2h 00m |

## Pre-requis

- Maitrise d'au moins un langage de programmation (C#, Java, Python, Node.js, etc.)
- Familiarite avec les workflows Git
- Licence GitHub Copilot valide
- Environnement de developpement configure avec VS Code et Copilot active
