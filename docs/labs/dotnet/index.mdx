# Labs — Parcours .NET (ASP.NET Core)

Ce parcours contient les exercices pratiques pour le track **.NET ASP.NET Core**.

Chaque lab correspond a un module de la formation et s'appuie sur le meme repository ASP.NET Core.

| Lab | Module | Description |
|-----|--------|-------------|
| [Lab 01 — Always-On Instructions](./lab-01.md) | Custom Instructions | Configuration `copilot-instructions.md` |
| [Lab 02 — File-Based Instructions](./lab-02.md) | Custom Instructions | Fichiers `*.instructions.md` avec `applyTo` |
| [Lab 03 — Prompt Files](./lab-03.md) | Prompt Files | Creation de `/review-code` et `/generate-feature` |
| [Lab 04 — Skills](./lab-04.md) | Agent Skills | Skill test-runner avec dotnet test |
| [Lab 05 — Custom Agents](./lab-05.md) | Custom Agents | Agent Clean Architecture avec scope controle |
| [Lab 06 — End-to-End Workflow](./lab-06.md) | Projet Final | Environnement Copilot complet |
