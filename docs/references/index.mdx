## Les 5 mecanismes de personnalisation

| Mecanisme | Ce qu'il fait |
|-----------|---------------|
| Always-on instructions | Enseigne a Copilot vos standards et architecture — applique a chaque requete |
| File-based instructions | Applique des regles differentes selon les parties du codebase |
| Prompt files | Commandes slash reutilisables pour les taches recurrentes |
| Agent skills | Workflows automatises multi-etapes avec scripts |
| Custom agents | Personas IA specialisees avec scope et outils controles |

## Glossaire

| Terme | Definition |
|-------|-----------|
| `copilot-instructions.md` | Fichier d'instructions always-on inclus dans chaque requete |
| `*.instructions.md` | Fichier d'instructions file-based avec pattern `applyTo` |
| `.prompt.md` | Fichier prompt qui definit une commande slash reutilisable |
| `SKILL.md` | Fichier de description d'un agent skill |
| `@workspace` | Chat participant qui donne a Copilot le contexte du workspace |
| `@vscode` | Chat participant pour les commandes VS Code |
| `@terminal` | Chat participant pour le contexte du terminal |
