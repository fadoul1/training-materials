# Labs — Parcours Java (Spring Boot)

Ce parcours contient les exercices pratiques pour le track **Java Spring Boot**.

Chaque lab correspond a un module de la formation et s'appuie sur le meme repository Spring Boot.

| Lab | Module | Description |
|-----|--------|-------------|
| [Lab 01 — Always-On Instructions](./lab-01-always-on-instructions.md) | Custom Instructions | Configuration `copilot-instructions.md` |
| [Lab 02 — File-Based Instructions](./lab-02-file-based-instructions.md) | Custom Instructions | Fichiers `*.instructions.md` avec `applyTo` |
| [Lab 03 — Prompt Files](./lab-03-prompt-files.md) | Prompt Files | Creation de `/review-code` et `/generate-tests` |
| [Lab 04 — Skills](./lab-04-skills.md) | Agent Skills | Skill test-runner avec scripts Maven |
| [Lab 05 — Custom Agents](./lab-05-custom-agents.md) | Custom Agents | Agent de refactoring avec scope controle |
| [Lab 06 — End-to-End Workflow](./lab-06-end-to-end-workflow.md) | Projet Final | Environnement Copilot complet |
